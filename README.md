# Amazon Sales Analysis Dashboard | Power BI Project

## Project Overview

The Amazon Sales Analysis Dashboard is a data visualization project developed using Power BI to analyze e-commerce sales performance. The dashboard focuses on understanding sales trends, product performance, and geographic distribution of orders.

The goal of this project is to convert raw transactional data into interactive visual insights that help evaluate business performance and identify growth opportunities.

## Objectives

The objectives of this project include:

* Analyze overall sales performance.
* Identify high-performing product categories.
* Understand regional sales distribution.
* Track product demand across different sizes and categories.
* Monitor sales trends over time.

## Tools and Technologies Used

* Power BI
* Data Cleaning and Transformation
* DAX Measures
* Data Modeling
* Interactive Data Visualization

## Dataset Description

The dataset used in this project contains Amazon e-commerce transaction data including the following fields:

* Order ID
* Order Date
* Category
* Size
* Quantity
* Amount
* Fulfilment Type
* Ship City
* Ship State
* Ship Country
* Courier Status

The dataset represents order-level sales information across multiple locations.

## Key Dashboard Features

### Sales Overview

Displays key metrics such as total sales, total orders, total quantity sold, and geographic coverage.

### Category Performance

Analyzes how different product categories contribute to total sales.

### Product Size Analysis

Examines the demand for products based on size.

### Regional Sales Analysis

Shows how sales are distributed across states and cities.

### Sales Trend Analysis

Visualizes sales growth and patterns over time.

## Dashboard Pages

### Page 1 – Sales Overview

Provides a high-level summary of sales performance using KPI cards and category analysis.

### Page 2 – Product Analysis

Focuses on category performance, product size demand, and quantity distribution.

### Page 3 – Regional Sales Analysis

Displays geographic sales performance across states and cities using bar charts and maps.

## Insights Gained

* Some product categories generate a larger share of total revenue.
* Sales performance varies across different geographic regions.
* Product size plays a role in demand patterns.
* Sales trends help identify peak purchasing periods.

## Conclusion

This dashboard demonstrates the use of Power BI to analyze e-commerce sales data and generate actionable insights. The project highlights skills in data preparation, visualization, and analytical storytelling using business intelligence tools.
